## Grammar of the Thrice-Nine Language

### Three Cases

```
1. NOMINATIVE (Who? What?)
   Suffix: -A
   Example: Ivan-A (Ivan)

2. ACTIVE (What does?)
   Suffix: -E
   Example: Ivan-E (Ivan does)

3. DIRECTIONAL (Where to? Towards what?)
   Suffix: -O
   Example: Ivan-O (towards Ivan)
```

### Three Tenses

```
1. PAST (was)
   Prefix: B-
   Example: B-walk = walked

2. PRESENT (is)
   No changes
   Example: walk = walks

3. FUTURE (will be)
   Prefix: G-
   Example: G-walk = will walk
```

### Three Numbers

```
1. SINGULAR (one)
   No suffix
   Example: warrior

2. DUAL (two)
   Suffix: -V
   Example: warrior-V (two warriors)

3. PLURAL (three or more)
   Suffix: -G
   Example: warrior-G (three+ warriors)
```

## Code in the Thrice-Nine Kingdom Language

```vibee
// ===================================================================
// TRIDEVYATITSA IN CODE
// ===================================================================

/// 27 alphabet symbols
const TRIDEVYATITSA: [char; 27] = [
    'A', 'B', 'G', 'D', 'E', 'F', 'Z', 'H', 'I',  // Copper (1-9)
    'K', 'L', 'M', 'N', 'O', 'P', 'R', 'S', 'T',  // Silver (10-18)
    'U', 'V', 'W', 'X', 'Y', 'Z', 'Q', 'J', 'Y',  // Golden (19-27)
];

/// Numerical value of a letter
fn value(letter: char) -> Option<i32> {
    TRIDEVYATITSA.iter()
        .position(|&l| l == letter)
        .map(|i| (i + 1) as i32)
}

/// Numerical value of a word
fn word_value(word: &str) -> i32 {
    word.chars()
        .filter_map(value)
        .sum()
}

/// Three kingdoms
enum Kingdom {
    Copper,      // letters 1-9
    Silver,      // letters 10-18
    Golden,      // letters 19-27
}

fn letter_kingdom(letter: char) -> Option<Kingdom> {
    value(letter).map(|v| match v {
        1..=9 => Kingdom::Copper,
        10..=18 => Kingdom::Silver,
        19..=27 => Kingdom::Golden,
        _ => unreachable!(),
    })
}

/// Convert number to ternary notation using letters
fn to_tridevyatitsa(mut number: i32) -> String {
    let mut result = String::new();
    while number > 0 {
        let remainder = ((number - 1) % 27) as usize;
        result.insert(0, TRIDEVYATITSA[remainder]);
        number = (number - 1) / 27;
    }
    result
}

// Examples:
// word_value("GGG") = 3 + 3 + 3 = 9
// to_tridevyatitsa(27) = "Yat"
// to_tridevyatitsa(999) = "AII" (1x27^2 + 9x27 + 9 = 729 + 243 + 27 = 999)
```
