### Функции скрипта:

1. **Параллельное тестирование**
   - Использует GNU parallel для ускорения
   - Настраиваемое количество потоков
   - Прогресс-бар в реальном времени

2. **Красивый вывод**
   - Цветовая кодировка (красный, зелёный, жёлтый, голубой, пурпурный, циан)
   - Эмодзи для визуализации
   - Unicode box drawing characters

3. **Генерация отчётов**
   - Текстовый отчёт: `generated_code_test_report_parallel.txt`
   - JSON отчёт: `generated_code_test_report_parallel.json`

4. **Метрики**
   - Всего файлов / успешно / неудачно
   - Всего тестов / прошло / не прошло
   - Общее время / среднее время на файл
   - Качество генерации кода (отлично/хорошо/удовлетворительно/плохо)

### Использование:

```bash
# Использовать 8 потоков (рекомендуется)
./scripts/test_generated_code_parallel.sh 8

# Использовать 4 потока
./scripts/test_generated_code_parallel.sh 4

# Использовать все ядра (авто)
./scripts/test_generated_code_parallel.sh
```
