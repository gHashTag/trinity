# Trinity Node - Desktop Application Architecture

## Overview

Trinity Node is a cross-platform desktop application that allows users to contribute CPU compute to the Trinity Network and earn $FPGA tokens.

## Core Components

### 1. Inference Engine

```rust
// src-tauri/src/inference/engine.rs

pub struct InferenceEngine {
    model: TernaryModel,
    config: InferenceConfig,
    stats: InferenceStats,
}

impl InferenceEngine {
    /// Load model shards from disk
    pub async fn load_model(&mut self, model_id: &str) -> Result<()>;
    
    /// Process inference request
    pub async fn infer(&self, request: InferenceRequest) -> Result<InferenceResponse>;
    
    /// Get current throughput (tokens/sec)
    pub fn get_throughput(&self) -> f64;
    
    /// Detect CPU capabilities (AVX2, AVX-512, NEON)
    pub fn detect_simd() -> SimdCapability;
}

pub struct InferenceConfig {
    pub max_batch_size: usize,
    pub max_sequence_length: usize,
    pub num_threads: usize,
    pub memory_limit_mb: usize,
}
```

### 2. Network Client

```rust
// src-tauri/src/network/client.rs

pub struct NetworkClient {
    peer_id: PeerId,
    swarm: Swarm<TrinityBehaviour>,
    scheduler_addr: Multiaddr,
}

impl NetworkClient {
    /// Connect to Trinity Network
    pub async fn connect(&mut self) -> Result<()>;
    
    /// Register as compute provider
    pub async fn register_node(&self, capabilities: NodeCapabilities) -> Result<NodeId>;
    
    /// Receive job from scheduler
    pub async fn receive_job(&mut self) -> Result<InferenceJob>;
    
    /// Submit completed job
    pub async fn submit_result(&self, job_id: JobId, result: InferenceResult) -> Result<()>;
    
    /// Report node status (heartbeat)
    pub async fn heartbeat(&self, stats: NodeStats) -> Result<()>;
}
```

### 3. Wallet Manager

```rust
// src-tauri/src/wallet/manager.rs

pub struct WalletManager {
    keypair: Keypair,
    address: Address,
    balance: Balance,
}

impl WalletManager {
    /// Create new wallet
    pub fn create() -> Result<Self>;
    
    /// Import from seed phrase
    pub fn from_mnemonic(mnemonic: &str) -> Result<Self>;
    
    /// Get $FPGA balance
    pub async fn get_balance(&self) -> Result<Balance>;
    
    /// Withdraw to external address
    pub async fn withdraw(&self, to: Address, amount: u64) -> Result<TxHash>;
    
    /// Export seed phrase (encrypted)
    pub fn export_mnemonic(&self, password: &str) -> Result<EncryptedMnemonic>;
}
```
