# Firebird Icons

Place the following icon files here:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Design Guidelines

- Primary color: #f39c12 (Firebird orange)
- Background: Transparent or #1a1a2e (dark)
- Symbol: Fire/Phoenix bird or flame

## Temporary Icons

For testing, you can use placeholder icons. Generate them with:

```bash
# Using ImageMagick
convert -size 16x16 xc:#f39c12 icon16.png
convert -size 48x48 xc:#f39c12 icon48.png
convert -size 128x128 xc:#f39c12 icon128.png
```
